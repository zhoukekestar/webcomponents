# chart-js

# Install
`<link rel="import" href="chart-js/index.html">`

# Create
```html
<chart-js labels='["jan", "feb", "mar"]' datasets='[{"label": "first", "data": [56, 4, 89]}, {"label": "second", "data": [89, 60, 12]}, {"label": "third", "data": [50, 80, 50]}, {"label": "4th", "data": [60, 20, 60]}]'></chart-js>
```

# Attributes
* `labels`
* `datasets`
